## Workshop Program

This page contains the schedule and program for the Humanities Data and Mapping Environments workshop at ESU DH – Besançon 2026.

The detailed program will be published here before the workshop begins.

### Topics Covered

The workshop will address the following themes:

- Introduction to humanities data and spatial thinking
- Fundamentals of geographic information systems (GIS)
- Creating and cleaning geospatial data from humanistic sources
- Web mapping tools and platforms
- Critical cartography and reflexive mapping practices
- Collaborative and open data environments

More details will be added as the program is finalized.
