## Before the Workshop

This page contains information and resources to help participants prepare for the Humanities Data and Mapping Environments workshop.

### Software to Install

Participants are encouraged to install the following tools before the workshop begins:

- A modern web browser (Chrome or Firefox recommended)
- A text editor (e.g., [VS Code](https://code.visualstudio.com/))
- [QGIS](https://qgis.org/) – open-source geographic information system
